# astro-up Scoop Bucket

[Scoop](https://scoop.sh/) bucket for [astro-up](https://github.com/astro-up/astro-up) — astrophotography software update manager for Windows.

## Usage

```powershell
scoop bucket add astro-up https://github.com/astro-up/scoop-bucket
scoop install astro-up
```

## Update

```powershell
scoop update astro-up
```

## About

This bucket is automatically updated on each astro-up release via GitHub Actions.
